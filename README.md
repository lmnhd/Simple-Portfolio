# Simple Portfolio

A clean, standalone Next.js portfolio website focused on displaying projects and resume information. Extracted from the Lyrical project.

## Features
- Professional Experience Timeline
- Featured Projects Showcase
- Skills Grid
- Resume PDF Download
- Responsive Design with Tailwind CSS

## Architecture
- **Framework**: Next.js 14/15
- **Styling**: Tailwind CSS + Shadcn UI
- **Language**: TypeScript

## Getting Started

```bash
npm install
npm run dev
```
